AI_CHABOT

This project is a personal AI assistant built using a self-hosted Large Language Model (LLM) with vector-based knowledge retrieval. It provides a chatbot interface that answers user queries using stored contextual information.

Features:

-Self-hosted LLM using Ollama (LLaMA2)
-Vector database using Pinecone
-Retrieval Augmented Generation (RAG)
-FastAPI backend
-Simple web-based chatbot UI
-Context-aware responses

---

Technology used:

-Python
-FastAPI
-Ollama (LLaMA2)
-Pinecone
-Sentence Transformers
-HTML, JavaScript
